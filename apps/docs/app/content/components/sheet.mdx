## Usage

```tsx
import Sheet from '@/components/ui/sheet'
```
```tsx
<Sheet.Root>
  <Sheet.Trigger>Open Sheet</Sheet.Trigger>
  <Sheet.Portal>
    <Sheet.Overlay />
    <Sheet.Content>
      <Sheet.Handle />
      <Sheet.Header>
        <Sheet.Title>Sheet Title</Sheet.Title>
        <Sheet.Description>
          This is a description of the sheet.
        </Sheet.Description>
      </Sheet.Header>
      <div>
        {/* Sheet content goes here */}
      </div>
      <Sheet.Footer>
        <Sheet.Close asChild>
          <Button>Cancel</Button>
        </Sheet.Close>
        <Button>Continue</Button>
      </Sheet.Footer>
    </Sheet.Content>
  </Sheet.Portal>
</Sheet.Root>
```

## API Reference

### Sheet.Root
The root sheet element that wraps the entire component.

**Props:**
- `defaultOpen` - boolean - Default open state of the sheet
- `open` - boolean - Controls the open state of the sheet
- `onOpenChange` - function - Callback when the open state changes
- `modal` - boolean - Whether the sheet is modal (default: true)
- `container` - HTMLElement - Container element (default: document.body)
- `onAnimationEnd` - function - Callback when animation ends
- `dismissible` - boolean - Whether the sheet can be dismissed (default: true)
- `handleOnly` - boolean - Whether only the handle can be used to drag (default: false)
- `repositionInputs` - boolean - Whether to reposition inputs (default: true)

**Snap Points Props:**
- `snapPoints` - array - Array of snap points
- `activeSnapPoint` - boolean - Currently active snap point
- `setActiveSnapPoint` - function - Function to set the active snap point
- `fadeFromIndex` - number - Index from which to fade
- `snapToSequentialPoint` - boolean - Whether to snap to sequential points

### Sheet.Trigger
The element that triggers the sheet to open.

**Props:**
- `asChild` - boolean - Whether to render as a child element (default: false)

### Sheet.Portal
Portals the sheet content to the document body. No props.

### Sheet.Overlay
The backdrop overlay behind the sheet.

**Props:**
- `asChild` - boolean - Whether to render as a child element (default: false)

### Sheet.Content
The main content container of the sheet.

**Props:**
- `asChild` - boolean - Whether to render as a child element (default: false)

### Sheet.Handle
Drag handle for dismissing the sheet. No props.

### Sheet.Header
Header section wrapper for title and description.

### Sheet.Footer
Footer section wrapper for actions.

### Sheet.Title
The sheet title element.

**Props:**
- `asChild` - boolean - Whether to render as a child element (default: false)

### Sheet.Description
The sheet description element.

**Props:**
- `asChild` - boolean - Whether to render as a child element (default: false)

### Sheet.Close
Button to close the sheet.

**Props:**
- `asChild` - boolean - Whether to render as a child element (default: false)
